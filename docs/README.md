# Job Portal API Documentation

This folder contains the **Postman Collection** for the Job Portal Project. It serves as the primary technical guide for Frontend and Mobile developers to integrate with the backend services.

## 📂 Directory Structure
* `job-portal-collection.json` — The raw Postman collection file for import.
* `README.md` — This documentation file.

---

## 🛠 How to Get Started
1. Open **Postman**.
2. Click the **Import** button and select the `job-portal-collection.json` file from this directory.
3. Set your environment variable `base_url` to `http://localhost:8080` (or your specific server URL).
